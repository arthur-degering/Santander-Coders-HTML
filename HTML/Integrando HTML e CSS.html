Tags de Formatação de texto

Negrito é representado pela tag < b>, que é apenas um destaque visual.
Há também a tag < strong>, que aponta que aquele trecho é importante para máquinas de pesquisa.
Para Italico, o comando é < I>, sendo apenas efeito visual. Para dar enfase, o comando é < em>, que também é importante para pesquisas.
Sublinhado se verifica com a tag < u>, mas não se recomenda usar por ser uma tag antiga; o recomendado é usar CSSbr
Para riscar um texto, utiliza-se a tag < s>
Para grifar o texto, a tag adequada é < mark>

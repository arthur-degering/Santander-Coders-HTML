Estrutura básica do arquivo html

Todas as tags devem ser abertas por < tag> e fechadas < /tag>, ignorando os espaços.

1) A ordem de tags para criar uma página é: html; head, que contem o title; body. Toda tag deve ser fechada.

2) A tag html define os atributos do site, como linguagem, que se define pela tag "lang=sigla da lingua" como pt-br

3) "Head" é o cabeçalho da página. Dentro desta tag, é necessário incluir o título com a tag "title"
Há também as tags "META", definidas pela documentação MDN

Viewport é a área de renderização do "body"

4) "Body" é a tag que compõe a página em si de forma genérica. Cada parágrafo é definido com a tag "p" dentro da "viewport"

Agora vamos tratar das principais tags HTML

1) Os títulos dentro do site são identificados em < h1>, indo de 1 a 6, diminuinto respectivamente de tamanho.

2) Definir parágrafo ocorre com a tag < p> e quebra de linha ocorre com < br>
Para criar uma separação em linha horizontal, é preciso usar a tag < hr>.

3) Para criar um link, como foi feito para apontar as tags metas, é necessário utilizar a tag < a href="link do site">texto para ser o hiperlink < /a>. Ao clickar no link, vai abrir o link na mesma página
Para abrir o link em outra aba, a tag é < a href="link do site" target="_blank">texto para ser o hiperlink < /a>

Imagens

Imagens é a única tag que não precisa fechar. Ela tem 2 atributos: source ("src") e Descrição Alternativa ("alt")


A tag é < img src="link" alt="descrição">, é possivel mudar o tamanho da imagem com os atributos "width" (largura) e height (altura). Se colocar apenas width, a altura será ajustada proporcionalmente.
Para a width, é possivel colocar o valor numérico em pixels ou em %; a fonte da imagem pode ser um link da internet.

Listas Numeradas

A lista não numerada usa duas tags: "ol" (ordered list) que indica o inicio de uma lista, e "li" (list item), que aponta cada item da lista com uma bolinha

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Listas Não Numeradas

A lista não numerada usa duas tags: "ul" (unordered list) que indica o inicio de uma lista, e "li" (list item), que aponta cada item da lista com uma bolinha.

  • Primeiro item
  • Segundo item
  • Terceirto item

Tags De Uso Genérico

Estas tags não alteram diretamente o documento. depende de algum fator externo, como codificação em CSS. Podem ser as tags "span", que mantem a linha (chamado de inline) e "div", que não mantem inline.

Para Italico, o comando é < I>, sendo apenas efeito visual. Para dar enfase, o comando é < em>, que também é importante para pesquisas.
Sublinhado se verifica com a tag < u>, mas não se recomenda usar por ser uma tag antiga; o recomendado é usar CSSbr
